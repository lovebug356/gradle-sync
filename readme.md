# Gradle-Sync

**Small utility to synchronize the gradle version with the cargo version.**

## Usage

Add build dependency in ```Cargo.toml```:

```toml
[build-dependencies]
gradle-sync = 0.1.0
```

Add the following code snipper to ```build.rs```:

```[rust]
```
